Exercice 3.1

Ecrire un algorithme qui demande un nombre à l’utilisateur, et l’informe ensuite si ce nombre est positif ou négatif (on laisse de côté le cas où le nombre vaut zéro).

Pseudocode:

Tester l'algorithme :

Sous le capot

                                    JAVASCRIPT
                                     
    function Exo_3_1_innerHTML() {
        // DEBUT
        //iNombre en numerique
        var iNombre;
        //DEBUT
        // ECRIRE "Donnez un nombre"
        // Lire le Nombre
        iNombre = document.getElementById("iNombreUtilisateur").value;
        // Si Nombre est supérieur à 0 alors
        if (iNombre > 0) {
            //  Ecrire le Nombre est positif
            document.getElementById("div_resultat_js").innerHTML = "le Nombre est positif !";
        }
        // sinonsi iNombre < 0 alors
        else if (iNombre < 0) {
            document.getElementById("div_resultat_js").innerHTML = "le Nombre est negatif !";
        }
        //Sinon
        //  Ecrire le Nombre n'est pas évalué
        else {
            document.getElementById("div_resultat_js").innerHTML = "le Nombre n'est pas évalué !!";
        }
        //Fin
    }
