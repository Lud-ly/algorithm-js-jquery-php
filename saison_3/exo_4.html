Exercice 3.4

Ecrire un algorithme qui demande un nombre à l’utilisateur, et l’informe ensuite si ce nombre est positif ou négatif (on inclut cette fois le traitement du cas où le nombre vaut zéro).

Pseudocode:

nombre en numerique debut ecrire nombre=prompt"donnez un nombre" lire le nombre si nombre >0 alors ecrire=alert le nombre est positif sinonsi nombre inferieur 0 alors ecrire=alert le nombre est negatif sinon ecrire=alert le nombre est 0 fin 

                                    JAVASCRIPT
                                     
                                        nombre = 0;
                                        console.log(nombre = prompt("donnez un nombre:"));
                                        if (nombre > 0) {
                                            alert("le nombre est positif")
                                        }
                                        else if (nombre < 0) {
                                            alert("le nombre est negatif");
                                        }
                                        else {
                                            alert("le nombre est 0");
                                        }
