Exercice 4.4

Un magasin de reprographie facture 0,10 E les dix premières photocopies, 0,09 E les vingt suivantes et 0,08 E audelà. Ecrivez un algorithme qui demande à l’utilisateur le nombre de photocopies effectuées et qui affiche la facture correspondante.

    function Exo_4_4_innerHTML() {
        //Variables nombre, prix en Numérique

        //Début
        nbrCopies = 0;
        prix = 0;
        //Ecrire "Nombre de photocopies : 
        //Lire nombre
        nbrCopies = document.getElementById("nbrCopies").value;
        //Si nombre <= 10 Alors
        // prix ← nombre * 0,1
        if (nbrCopies <= 10) {
            prix = nbrCopies * 0.1;
        }
        //Sinonsi nombre <= 30 Alors
        // prix ← nombre * 0,1 pour les 10 premieres plus nombre - 10  (soit 20) à 0.09
        else if (nbrCopies <= 30) {
            prix = 10 * 0.1 + (nbrCopies - 10) * 0.09;
        }
        //Sinon
        // prix ← nombre * 0,1 pour les 10 premieres plus 20 à 0.09 plus nombre-30(soit nombre au delà de 30) à 0.08
        else {
            prix = 10 * 0.1 + 20 * 0.09 + (nbrCopies - 30) * 0.08;
        }
        //Ecrire "Le prix total à payer est: ", prix
        document.getElementById("div_resultat_js").innerHTML = ("innerHTML le prix à payer est de " + prix.toFixed(2) + " €");

    }//fin
