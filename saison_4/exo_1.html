Exercice 4.1

Formulez un algorithme équivalent à l’algorithme suivant : Si Tutu > Toto + 4 OU Tata = "OK" Alors Tutu ← Tutu + 1 Sinon Tutu ← Tutu – 1 Finsi

Pseudocode:

ecrire entrez une valeur pour tutu
ecrire entrez une valeur pour toto
lire tutu et toto
Si Tutu <= Toto + 4 OU Tata <> "OK" Alors
Tutu ← Tutu + 1
Sinon
Tutu ← Tutu - 1
Finsi
fin

Tester l'algorithme :

Loi de Morgan

        function Exo_4_1_innerHTML() {
            // DEBUT
            //VARIABLE tutu en numerique
            //VARIABLE toto en numerique
            //VARIABLE sTata ="ok"
            var Tutu;
            var Toto;
            var sTata = "ok";
            //ecrire entrez une valeur pour tutu
            //ecrire entrez une valeur pour toto
            //tutu et toto
            Tutu = document.getElementById("iValeurTutu").value;
            Toto = document.getElementById("iValeurToto").value;

            //loi de morgan
            //Si Tutu <= Toto + 4 ET sTata <> "OK" Alors
            if (Tutu <= Toto + 4 && sTata) {
                // Tutu ← Tutu - 1
                Tutu -= 1;
            }
            // Sinon
            else {
                // Tutu ← Tutu + 1
                Tutu += 1;
            }
            document.getElementById("div_resultat_js").innerHTML = (Tutu);
            //fin
        }
