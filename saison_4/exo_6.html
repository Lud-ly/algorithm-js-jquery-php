Exercice 4.6

Les élections législatives, en Toonerie Nationale, obéissent à la règle suivante :  lorsque l'un des candidats obtient plus de 50% des suffrages, il est élu dès le premier tour.  en cas de deuxième tour, peuvent participer uniquement les candidats ayant obtenu au moins 12,5% des voix au premier tour. Vous devez écrire un algorithme qui permette la saisie des scores de quatre candidats au premier tour. Cet algorithme traitera ensuite le candidat numéro 1 (et uniquement lui) : il dira s'il est élu, battu, s'il se trouve en ballottage favorable (il participe au second tour en étant arrivé en tête à l'issue du premier tour) ou défavorable (il participe au second tour sans avoir été en tête au premier tour).

Pseudocode:

Variables trum, melencho, poutin, macro en Numérique
Variables Cas1, Cas2, Cas3, Cas4 en Booléen
Début
Ecrire "Entrez les scores des quatre prétendants :"
Lire trump, melenchon, poutine, macron
Cas1 ← trump > 50
Cas2 ← melenchon > 50 ou poutine > 50 ou macron > 50
Cas3 ← trump >= melencho et trump >= poutin et trump >= macro
Cas4 ← trump >= 12,5
Si Cas1 Alors
Ecrire “Elu au premier tour"
Sinonsi Cas2 ou Non(Cas4) Alors
Ecrire “Battu, éliminé, sorti !!!”
SinonSi Cas3 Alors
Ecrire "Ballotage favorable"
Sinon
Ecrire "Ballotage défavorable
" FinSi
Fin

                                        function Exo_4_6() {
                                            trum = 0; melencho = 0; poutin = 0; macro = 0;
                                            trum = (prompt("Entrez 1er prétendants :"));
                                            melencho = (prompt("Entrez 2eme prétendants :"));
                                            poutin = (prompt("Entrez 3eme prétendants :"));
                                            macro = (prompt("Entrez 4eme prétendants :"));
                            
                            
                                            cas1 = trum > 50;
                                            cas2 = melencho > 50 && poutin > 50 && macro > 50;
                                            cas3 = trum >= melencho && trum >= poutin && trum >= macro;
                                            cas4 = trum >= 12.5;
                            
                                            for (i = 0; i <= 4; i++) {
                                                if (cas1) {
                                                    alert("Elu au premier tour");
                                                }
                                                else if (cas2 || cas4 == false) {
                                                    alert("Battu, éliminé, sorti !!!");
                                                }
                                                else if (cas3) {
                                                    alert("Ballotage Favorable");
                                                }
                                                else {
                                                    alert("Ballotage Defavorable");
                                                }
                                            }
                                        }
