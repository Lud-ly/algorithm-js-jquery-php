Exercice 5.11

Écrire un algorithme qui permette de connaître ses chances de gagner au tiercé, quarté, quinté et autres impôts volontaires. On demande à l’utilisateur le nombre de chevaux partants, et le nombre de chevaux joués. Les deux messages affichés devront être : Dans l’ordre : une chance sur X de gagner Dans le désordre : une chance sur Y de gagner X et Y nous sont donnés par la formule suivante, si n est le nombre de chevaux partants et p le nombre de chevaux joués (on rappelle que le signe ! signifie "factorielle", comme dans l'exercice 5.7 ci-dessus) : X = n ! / (n - p) ! Y = n ! / (p ! * (n – p) !) NB : cet algorithme peut être écrit d’une manière simple, mais relativement peu performante. Ses performances peuvent être singulièrement augmentées par une petite astuce. Vous commencerez par écrire la manière la plus simple, puis vous identifierez le problème, et écrirez une deuxième version permettant de le résoudre.

    function Exo_5_11_innerHTML() {
        //Variables nbChevauxPart, nbChevauxJoue, i, ordre, desordre en Numérique
        var nbChevauxPart;
        var nbChevauxJoue;
        var i;
        var ordre;
        var desordre;
        // Debut
        //  Ecrire "Entrez le nombre de chevaux partants : "
        //Lire nbChevauxPart
        parseInt(nbChevauxPart = document.getElementById("nbChevauxPart").value);


        //Ecrire "Entrez le nombre de chevaux joués : "
        // Lire nbChevauxJoue
        parseInt(nbChevauxJoue = document.getElementById("nbChevauxJoue").value);

        ordre = 1
        desordre = 1
        //Pour i ← 1 à nbChevauxJoue
        for (i = 1; i <= nbChevauxJoue; i++) {
            ordre = ordre * (i + nbChevauxPart - nbChevauxJoue)
            desordre = desordre * i;
        }
        //Ecrire "Dans l’ordre, une chance sur ", ordre
        //Ecrire "Dans le désordre, une chance sur ", ordre / desordre
        document.getElementById("div_resultat_ordre").innerHTML = ("innerHtml Dans l’ordre, une chance sur " + ordre);
        document.getElementById("div_resultat_desordre").innerHTML = ("Dans le désordre, une chance sur " + ordre / desordre);
    } //Fin
