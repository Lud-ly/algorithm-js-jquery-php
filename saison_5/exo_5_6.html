ALGORITHME ET PSEUDO-CODE:

Exercice 5.6

Ecrire un algorithme qui demande un nombre de départ, et qui calcule la somme des entiers jusqu’à ce nombre. Par exemple, si l’on entre 5, le programme doit calculer : 1 + 2 + 3 + 4 + 5 = 15 NB : on souhaite afficher uniquement le résultat, pas la décomposition du calcul.

Pseudocode:

Tester l'algorithme :

Sous le capot

                                    JAVASCRIPT
                                     
        function Exo_5_6_innerHTML() {
            //Variables iNombre, i, Somme en Entier
            var iNombre;
            var i;
            var iSomme;
            // Debut
            //Ecrire "Entrez un nombre : "
            //Lire iNombre
            iNombre = parseInt(document.getElementById("iNombreUtilisateur").value);
            //Somme = 0
            iSomme = 0;
            //Pour i = 1 à iNombre
            for (i = 0; i <= iNombre; i++) {
                // iSomme ← iSomme + i
                iSomme = iSomme + i;
            }//fin pour
            //Ecrire "La somme est : ", Somme
            document.getElementById("div_resultat_js").innerHTML = ("innerHTML La somme est : " + iSomme);
        }//fin
