Exercice 5.1

Ecrire un algorithme qui demande à l’utilisateur un nombre compris entre 1 et 3 jusqu’à ce que la réponse convienne.

Pseudocode:

VARIABLE iNombreUtilisateur en NUMERIQUE
DEBUT
ECRIRE "Entre un nombre entre 1 et 3"
LIRE iNombreUtilisateur
TANTQUE ((iNombreUtilisateur < 1) || (iNombreUtilisateur> 3))
ECRIRE "Veuillez ressaisir un nombre entre 1 et 3 svp"
LIRE iNombreUtilisateur
FINTANTQUE
ECRIRE "Bravo, tu as réussi !"
FIN

Tester l'algorithme :

         

Voir les Codes

                                     
    function Exo_5_1()
    {
    // DEBUT
        // VARIABLE iNombreUtilisateur en NUMERIQUE
        var iNombreUtilisateur;
        
        // ECRIRE "Entrez un nombre entre 1 et 3"
        // LIRE iNombreUtilisateur
        iNombreUtilisateur= parseInt(prompt("Entrez un nombre entre 1 et 3"));
        
        // TANTQUE ((iNombreUtilisateur < 1) || (iNombreUtilisateur > 3))
        while ((iNombreUtilisateur < 1) || (iNombreUtilisateur > 3))
        {
            // ECRIRE "Veuillez ressaisir un nombre entre 1 et 3 svp"
            // LIRE iNombreUtilisateur
            iNombreUtilisateur= parseInt(prompt("Entrez un nombre entre 1 et 3"));
        // FINTANTQUE
        }
        
        // ECRIRE "Bravo, tu as réussi !"
        alert("Bravo, tu as réussi !");

    // FIN
