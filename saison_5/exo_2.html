Exercice 5.2

b) Ecrire un algorithme qui choisit un nombre au hasard compris entre 1 et 100. Pour ceci, on utilise la fonction : iNombreHasard <- random[1..100] Demander à l'utilisateur de trouver le nombre. On fera apparaître un message : « Plus petit ! », et inversement, « Plus grand ! » jusqu’à ce que l'utilisateur ait trouvé et afficher en combien de coups.

Pseudocode:

VARIABLE iNombreHasard, iRepUtilisateur, iNombreCoup en NUMERIQUE
DEBUT
iNombreCoup= 1
iNombreHasard= random[1..100]
ECRIRE "Trouvez le nombre mystère, compris entre 1 et 100"
LIRE iRepUtilisateur
TANT QUE iRepUtilisateur != iNombreHasard
iNombreCoup++
SI (iRepUtilisateur inferieur iNombreHasard) ALORS
ECRIRE "Plus grand, essayez à nouveau :"
LIRE iRepUtilisateur SINON ECRIRE "Plus petit, essayez à nouveau :" LIRE
iRepUtilisateur FINSI FINTANTQUE ECRIRE "Bravo, vous avez trouvé en " + iNombreCoup + " coup(s)."
FIN

Trouvez le nombre mystère :

        // ALGORITHME Exo_5_2b_innerHTML
        function Exo_5_2b_innerHTML()
        {
        // DEBUT
            // VARIABLE iNombreHasard, iRepUtilisateur, iNombreCoup en NUMERIQUE
            var iRepUtilisateur;
            
            // ECRIRE "Trouvez le nombre mystère, compris entre 1 et 100"
            // LIRE iRepUtilisateur
            iRepUtilisateur= document.getElementById("iRepUtilisateur").value;
            
            // TANT QUE iRepUtilisateur != iNombreHasard
            iNombreCoup++;
            if (iRepUtilisateur != iNombreHasard)
            {
                // SI (iRepUtilisateur < iNombreHasard) ALORS
                if (iRepUtilisateur < iNombreHasard)
                {
                    // ECRIRE "Plus grand, essayez à nouveau :"
                    // LIRE iRepUtilisateur
                    document.getElementById("div_resultat_js").innerHTML= "Plus grand, essayez à nouveau.";
                }
                else
                {
                // SINON
                    // ECRIRE "Plus petit, essayez à nouveau :"
                    // LIRE iRepUtilisateur
                    document.getElementById("div_resultat_js").innerHTML= "Plus petit, essayez à nouveau.";
                }
                // FINSI
            }
            else
            {
                // ECRIRE "Bravo, vous avez trouvé en " + iNombreCoup + " coup(s)."
                document.getElementById("div_resultat_js").innerHTML= "Bravo, vous avez trouvé en " + iNombreCoup + " coup(s).";
            }
            // FINTANTQUE
            
            document.getElementById("iNombreCoup").value= iNombreCoup;
            document.getElementById("iNombreHasard").value= iNombreHasard;
        // FIN	
        }
