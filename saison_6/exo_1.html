Exercice 6.1

Ecrire un algorithme qui déclare et remplisse un tableau de 7 valeurs numériques en les mettant toutes à zéro.

Pseudocode:

Tester l'algorithme :

Sous le capot

                                    JAVASCRIPT
                                     
    function Exo_6_1_innerHTML() {
        //Tableau stuff[6] en Numérique
        var stuff = [,,,,,,,];
        var i;
        //Variable i en Numérique
        //Debut
        //Pour i ← 0 à 6
        for (i = 0; i < stuff.length; i++) {
            //stuff(i) ← 0
            stuff[i] = "[ " + 0 + " ]";
        } 
        document.getElementById("div_resultat_js").innerHTML = ("innerHTML"+stuff);
    }//Fin
