Exercice 6.13

Ecrivez un algorithme permettant, toujours sur le même principe, à l’utilisateur de saisir un nombre déterminé de valeurs. Le programme, une fois la saisie terminée, renvoie la plus grande valeur en précisant quelle position elle occupe dans le tableau. On prendra soin d’effectuer la saisie dans un premier temps, et la recherche de la plus grande valeur du tableau dans un second temps.

Pseudocode:

Variables Nb, Nbpos, Nbneg en Numérique
Variables global Tableau T() ,i en Numérique
Debut
Ecrire "Entrez le nombre n° ", i + 1
Lire T(value)
Push value dans T
Si longueur de T =5 alors
Ecrire tableau de Notes T
FinSi
pacourir valeur de T
Si val superieur nombre max
max = val;
pos = y;
Ecrire le plus grand est : " + max, la position du nombre est : " + pos
Fin

Tester l'algorithme :

Entrer des notes

Sous le capot

    var i = 2;
    var T = [];

    function Exo_6_13_innerHTML() {
        // Variables Nb, Nbpos, Nbneg en Numérique
        //Variables global Tableau T() ,i en Numérique
        var max = 0;
        var y = 0;
        var pos = 0;
        //Debut
        //  Ecrire "Entrez le nombre n° ", i + 1 
        document.getElementById("div_resultat").innerHTML = ("Entrer la notes n°" + [i++]);
        // Lire T(value)
        //Push value dans T
        T.push(parseInt(document.getElementById("MyInput").value));
        //Si longueur de T =5 alors
        if (T.length == 5) {
            //Ecrire tableau de Notes T
            document.getElementById("div_resultat").innerHTML = T.join(" , ");
            document.getElementById("div_resultat_js").innerHTML = ("innerHTML Voici le nouveau tableau : " + T);
        }//FinSi
        //pacourir valeur de T
        T.forEach(function (val) {
            //Si val superieur nombre max
            //  max = val;
            //pos = y;
            if (val > max) {
                max = val;
                pos = y;
            }
            y = y + 1;

        });
        //Ecrire le plus grand est : " + max, la position du nombre est : " + pos
        document.getElementById("div_result").innerHTML = ("innerHTML le nombre le plus grand est : " + max);
        document.getElementById("div_resultat_js").innerHTML = ("la position du nombre est : " + pos);
    }// Fin
