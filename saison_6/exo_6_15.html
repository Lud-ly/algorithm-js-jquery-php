Exercice 6.15

Pour sa naissance, la grand-mère de TITI place une somme de 1000 Euros sur son compte épargne rémunéré au taux de 2.75% (chaque année le compte est augmenté de 2.75%). Développer un algorithme permettant de stocker dans un tableau sur 20 ans associant à chaque anniversaire de TITI la somme acquise sur son compte. Puis, la grand-mère de TITI aura la possibilité de saisir un âge (compris entre 1 et 20 ans) et l'algorithme affichera la somme correspondante qu'il y'aura alors sur le compte.

    function Exo_6_15_innerHTML() {
        //Variables compte=1000,taux=2.75,interet,année,dateAnnif en numérique
        //Array tabCompt en numérique
        var compte = 1000;
        var taux = 2.75;
        var interet = 0;
        var tabCompt = [];
        var année = 1;
        var dateAnnif = 0;
        var soldYear = 0;
        //Début

        //Calcul des intéret
        interet = (compte * taux) / 100;
        //Ecrire Consulter le solde d'une année  1 à 20
        //Lire soldYear
        soldYear = parseInt(document.getElementById("year").value);
        //Pour année de 0 inferieur 20
        while (année <= 20) {
            //Ajout des intérets sur le compte
            compte += interet;
            //Ajouter valeurs dans le tableau
            tabCompt.push(année + "EME année" + " solde " + compte + " € " + " 
  ");
            dateAnnif = année;
            année++;
            //Si soldYear = dateAnnif
            if (soldYear == dateAnnif) {
                //Ecrire l'année et le solde
                document.getElementById("div_resultat").innerHTML = ((année - 1) + " ème Anniversaire " + "innerHTML Solde de Titi : " + compte + " € ");
            }//FinSi
        }//année suivant

        //Ecrire Solde de depart : " + compte
        //   document.getElementById("div_resultat_js").innerHTML = ("Solde de depart : " + compte + " €");
        //Ecrire Tableau tabCompt
        document.getElementById("div_result").innerHTML = (tabCompt);
    }//Fin
