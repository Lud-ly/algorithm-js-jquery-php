ALGORITHME ET PSEUDO-CODE:

Exercice 6.4

Que produit l’algorithme suivant ? Tableau Nb(5) en Entier Variable i en Entier Début Pour i ← 0 à 5 Nb(i) ← i * i i suivant Pour i ← 0 à 5 Ecrire Nb(i) i suivant Fin Peut-on simplifier cet algorithme avec le même résultat ?

Pseudocode:

Tableau Nb(5) en Entier
Variable i en Entier
Début
Pour i ← 0 à 5
Nb(i) ← i * i
nb[i] = i * i;
i suivant
Pour i ← 0 à 5

Tester l'algorithme :

Sous le capot

                                    JAVASCRIPT
                                     
    function Exo_6_4_innerHTML() {
        // Tableau Nb(5) en Entier
        //Variable i en Entier
        var nb = [];
        var i = 0;
        // Début
        // Pour i ← 0 à 5
        for (i = 0; i <= 5; i++) {
            //Nb(i) ← i * i
            nb[i] = i * i;
            document.getElementById("div_resultat_js").innerHTML = ("innerHTML resultat" + nb);
            // i suivant
            // Pour i ← 0 à 5
        }
    }
