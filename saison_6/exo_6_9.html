Ecrivez un algorithme calculant la somme des valeurs d’un tableau (on suppose que le tableau a été préalablement saisi).

Tester l'algorithme :

Calcul de la somme des éléments d'un tableau :

Sous le capot

                                    JAVASCRIPT
                                     
    function Exo_6_9_innerHTML() {
        //Variables i,  Som ← 0, N en Numérique
        //Tableau T(N) en Numérique
        var N;
        var som = 0;
        var tableau = [1, 56, 8, 6, 5, 4, 7, 8, 2, 3, 2];
        //Debut
        //  Pour i ← 0 à longueur du Tableau
        for (var i = 0; i < tableau.length; i++) {
            //  Som ← Som + T(i)
            //i Suivant
            som = som + tableau[i];
        }
        //  Ecrire "Les éléments du tableau sont : : ", Tableau
        //Ecrire "La Somme des éléments est : ", Som
        document.getElementById("div_resultat").innerHTML = ("innerHTML Les éléments du tableau sont: " + tableau);
        document.getElementById("div_resultat_js").innerHTML = ("Somme des éléments est de  : " + som);
    }// Fin
