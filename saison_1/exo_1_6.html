ALGORITHME ET PSEUDO-CODE:

Exercice 1.6

Ecrire un pseudo-code permettant d’échanger les valeurs de deux variables A et B, et ce quel que soit leur contenu préalable.

Pseudocode:

                                    JAVASCRIPT
                                     
    function Exo_1_6() {
        var a = parseInt(prompt("Saisir la valeur de a : "));
        var b = parseInt(prompt("Saisir la valeur de b : "));
        alert("a = "+ a + " b = "+ b);
        a = a + b
        b = a - b
        a = a - b
        alert("new a = "+ a + " new b = "+ b);
    }
