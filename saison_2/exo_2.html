ALGORITHME ET PSEUDO-CODE:

Exercice 2.2

Ecrire un programme qui demande un nombre à l’utilisateur, puis qui calcule et affiche le carré de ce nombre.

Pseudocode:

Variables nombre, carré en Numérique
Début
Ecrire "Entrez un nombre :"
Lire nombre
carré ← nombre * nombre
Ecrire "Son carré est : ", carré
Fin

Tester l'algorithme :

         

Voir les Codes

                                    JAVASCRIPT
                                     
                                        function Exo_2_2_innerHTML() {
                                            // Variables nombre, carré en Numérique
                                            var nombre;
                                            var carré;
                                            //Début
                                            //Ecrire "Entrez un nombre :"
                                            //Lire nombre
                                            nombre = document.getElementById("iNombreUtilisateur").value;
                                            // carré ← nombre * nombre
                                            carré = nombre * nombre;
                                            //Ecrire "Son carré est : ", carré
                                            document.getElementById("div_resultat_js").innerHTML = ("Bravo innerHTML Son carré est : " + carré)
                            
                                            //Fin
                                        }
