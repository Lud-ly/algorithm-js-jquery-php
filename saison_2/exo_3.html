Exercice 2.3

Ecrire un programme qui lit le prix HT d’un article, le nombre d’articles et le taux de TVA, et qui fournit le prix total TTC correspondant. Faire en sorte que des libellés apparaissent clairement.

Pseudocode:

Variables nbArticle,TauxTva,PrixHt,prixTtc en Numerique; Début Ecrire "Entrez le prix du produit hors taxes : Lire PrixHt Ecrire "Entrez le nombre d’articles :" Lire nbArticle Ecrire "Entrez le taux de TVA :" Lire TauxTva prixTtc = nbArticle * PrixHt * (1+TauxTva) Ecrire "Le prix toutes taxes est : ", pttc Fin

Tester l'algorithme :

                                    JAVASCRIPT
                                     
                                        function Exo_2_3_innerHTML() {
                                            // Variables nbArticle,TauxTva,PrixHt,prixTtc en Numerique;
                                            var nbArticle, TauxTva, PrixHt, PrixTtc;
                                            // Début
                                            //Ecrire "Entrez le prix du produit hors taxes :
                                            // Lire PrixHt
                                            PrixHt = document.getElementById("PrixHt").value;
                                            //Ecrire "Entrez le nombre d’articles :"
                                            //Lire nbArticle
                                            nbArticle = document.getElementById("nbArticle").value;
                                            //Ecrire "Entrez le taux de TVA :"
                                            //Lire TauxTva
                                            TauxTva = parseFloat(document.getElementById("TauxTva").value);
                                            // prixTtc = nbArticle * PrixHt * (1 + TauxTva/100);
                                            PrixTtc = nbArticle * PrixHt * (1 + TauxTva / 100);
                                            //Ecrire "Le prix toutes taxes est : ", pttc
                                            document.getElementById("div_resultat_js").innerHTML = ("innerHTML Le prix toutes taxes est : " + PrixTtc.toFixed(2) + "€");
                                            //Fin
                                        }
